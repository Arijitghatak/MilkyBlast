# 🚀 MilkyBlast

MilkyBlast is a 2D space shooting game built with Java where the player controls a spaceship, shoots incoming enemy rockets, and survives as long as possible in a fast-paced arcade-style environment.

## 🎮 Features
- 2D spaceship movement
- Enemy rocket spawning
- Shooting mechanics
- Collision detection
- Real-time game loop
- Simple arcade gameplay

## 🛠️ Tech Stack
- Java
- Java 2D Graphics (Swing/AWT)
- Object-Oriented Programming (OOP)

## 🎯 Objective
Control your spaceship, destroy enemy rockets, and survive for as long as possible.
